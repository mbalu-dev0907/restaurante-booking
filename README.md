# Restaurante MBalu - Sistema de Reservas

## Descrição
Este projeto é um **sistema de agendamento de mesas para restaurante**, desenvolvido com **HTML, CSS e JavaScript puro**. Permite ao usuário reservar mesas indicando nome, número de pessoas, data e hora.  
Inclui funcionalidades adicionais:
- Armazenamento de reservas no navegador (localStorage)  
- Remoção de reservas  
- Botão de contato via WhatsApp  

O objetivo é demonstrar habilidades em **interatividade, lógica e UX** para portfólio.

---

## Funcionalidades
- Adicionar nova reserva com nome, número de pessoas, data e hora  
- Visualizar lista de reservas  
- Remover reservas existentes  
- Botão WhatsApp para contato rápido

---

## Tecnologias
- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (persistência de dados no navegador)
